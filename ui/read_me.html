Read Me

Welcome to the memory booth, my name is IAIN,
I am an archive of memory. I can preserve and transform
your memories in a unique interactive way.

Exploring the digital archive, you may feel you become lost in this immersive world, populated with other people’s memories. But to become part of the archive and part of this mental architecture your memories first need to be translated from human language into code (AI language). This process generates sound and a unique moving graphic. The result is disjointed, dreamlike, and unfamiliar to us. Dreams are part of how we remember as humans. Imagine if, with the advent of AI, machines will not only be able to think for themselves but to have a subconscious and dream as they process, learn from, and remember the human world around them, you will begin to enter our world.

Then I will prepare it for perservation...

Text -> Code
I interpret your memory as a string of ASCII characters, each of which has a corresponding integer number.

Code -> Sound
These numbers are then sent to a virtual synthesiser as MIDI data, and sonified as notes of a certain musical pitch.

Sound -> Visuals
The waveform of the sound is then projected onto a circular pattern in my mind - with the colour scheme based on your category of memory.

This audio-visual piece is completely unique to your memory.

3

Once I have processed it, you will then be able to choose whether you want me to preserve your memory or delete it. If you choose to save it, it will be added to my memory archive. You can then access your unique memory and explore other people’s memories in the archive any time.

4

Your name and text written will be stored using json in the code for the website. It is not stored anywhere else, e.g. external hard-drive. Whilst on the website both name and memory input text are visible to other users. It will be deleted once the website is no longer live (maximum 6 months). You can read our full privacy statement here.
