2

Then I will prepare it for perservation...

Text -> Code
I interpret your memory as a string of ASCII characters, each of which has a corresponding integer number.

Code -> Sound
These numbers are then sent to a virtual synthesiser as MIDI data, and sonified as notes of a certain musical pitch.

Sound -> Visuals
The waveform of the sound is then projected onto a circular pattern in my mind - with the colour scheme based on your category of memory.

This audio-visual piece is completely unique to your memory.

3

Once I have processed it, you will then be able to choose
whether you want me to preserve your memory or delete it.

If you choose to save it, it will be added to my memory archive.
You can then access your unique memory and explore other people’s
memories in the archive any time.

Thank you for using the memory booth, enjoy the journey!
